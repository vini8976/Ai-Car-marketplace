# AI Car Marketplace 🚗🧠

Built by **Vinita Prakash Singh** — a modern full stack AI-powered eCommerce platform to buy and sell cars.

## 🔧 Tech Stack
- **Frontend**: React 19, Next.js 15, Tailwind CSS, Shadcn UI
- **Backend**: Supabase, Prisma ORM
- **Auth**: Clerk Authentication
- **AI**: Gemini API (Google AI)
- **Security**: Arcjet
